1. File > New > Assembly
2. Подключаем в Reuse Library мою библиотеку "assembly"
3. Вставляем из "assembly" в сборку следующие детали: "Plita 7081-2202" и "Prokladka 7052-0052"
4. Запускаем мою прогу "setpaz.dll" через Ctrl+U
5. После запуска появляется менюшка выбора плоскости, выбираем на Plita, face с именем Top_5_0,
   посел выбора появляется еще одна менюшка, в ней выбираем весь объект Prokladka
6. UG-ешка повисает на пару секунд и деталь "Prokladka" устанавливается в паз "Plita"
